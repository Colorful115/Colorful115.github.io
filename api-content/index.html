工具”，可以轻松地帮助我们实现音画匹配的目的。14、变换、裁剪、变形、防抖动（鼠标操作）视频的属性有很多，这里我们重点来看一下变换、裁剪、变形和防抖动这四个特性。防抖动时FinalCutPro中非常强大的一个功能，它在默认情况下是不启动的。启动之后需要等待一会，因为软件在自己分析。可以在一定程度上有效地改善画面的抖动幅度。15、关键帧的概念和渐进运动关键帧可以理解为片段中“做下标记”的那一帧，两个以上的关键帧才能互相配合使用，实现某种运动或变化。将鼠标鼠标悬停在是拼检查器的这个位置，此时就自动出现了一个打关键帧的按钮，在这里点击就可以打一个关键帧，接着我们将时间轴拖移到下一个位置，然后再打一个关键帧，然后在这个关键帧的位置调整一下位置效果。看到上图我们就知道，第一个关键帧记录的片段属性和第二个关键帧记录的片段属性之间，它们有一个渐进运动的特性。关键帧的作用：就是记录片段属性，每个关键帧都可以记录一种属性。我们也可以在这里打关键帧，如下图：其实，片段的任何属性都可以单独来打关键帧，比如效果、颜色、音量等等。关键帧是我们实现渐近运动效果最重要的概念。16、速率和方向每一个片段都有它原始的速率和方向，在很多时候，我们需要加快或慢放片段的速率以及倒转片段的方向，从而呈现出丰富的、浪漫的、灵动的视觉效果。在FinalCutPro中，调整速率和方向的方式非常简单，用鼠标打开“重新定时选项”自定义速率：control+option+R注意：音频一般不能进行速率和方向的调整。否则音频会不可避免的变成噪音。所以，当我们针对片段做速率和方向的调整时候，要先分离他们的音频（control+shift+s），然后删掉它们，避免出现噪音。17、视频的效果效果：为视频片段添加额外的视觉效果，这些视觉效果可以提升或改善画面的原有特性，使之呈现出某种特定的偏向。首先，激活“效果浏览器窗口”，在效果窗口中我们可以看到，从上到下依次有12类视频效果，包括光源、冷热度、失真、拼贴、颜色、遮罩等等。每一类视频效果中都包含着众多具体的效果。选中其中一个效果，然后将它直接拖入这个片段，然后调整参数，size（大小）、Number（数量）、Speed（速度）、BlurAmount（涂抹数量）、Opacity（不透明度）等等特性。举几个例子：ProtectSlin（保护皮肤）、Shadows（阴影）、Brightness（亮度）、Scale（规模）、Contrast（对比）、Noise（噪音）等等特性。18、聚焦、抠像和遮罩FinalCutPro内置了一百多种效果，而这些效果中，这三种效果极为重要。第一种：“聚焦”，“聚焦”是“模糊”分类效果下的一种效果，其特性是可以制造出一个整体模糊而局部清晰的效果，从而实现某种程度的伪“景深”效果，弥补在画面拍摄中留下的遗憾或不足。除此之外，“聚焦”效果本身也可以极大地提升画面的通透感，凸显画面主体，营造艺术化氛围。可以拖移聚焦的位置及调整参数，Amount（数量）、Softness（柔和度）、Emphasis（重点）、Width（宽度）、Height（高度）。第二种：“抠像”，“抠像”分为两种具体的效果，一种是抠像器，另一种是亮度抠像器，抠像的本质是从画面中抠选出一部分，而将其它部分从画面中抹去。可以直接将“抠像器”拖入这个片段。在视频检查器中可以看到“样本颜色”的选项，我们点击这个样本颜色然后在画布中画出一个抹去的区域，这个区域代表一个选中的色彩范围，这时画面中所有的类似和接近的色彩都会被选中，这些色彩也都将被抹去在“显示”选项中，依次分别是复合、遮罩和原始状态。在进行抠像时，我们要激活“遮罩”状态，这时预览窗口的画布中，只显示白色和黑色，白色代表要留存的部分，而黑色代表要抹去的部分。完成抠像后，再激活“复合”状态，最后我们借助另外一个片段，将它放到这个片段的下方，来预览一下实际效果，可以看到，被抹去的那部分变成了透明，两个片段的画面融合在了一起。下面我们来看一下“亮度抠像器”：在亮度控制条中，左侧是黑色，右侧是白色。相对应的黑色代表画面中的暗部，白色代表画面中的亮部。控制轴向左拖移时，表示将画面中的亮部渐渐抹去。控制轴向右拖移表示将画面中的暗部渐渐抹去。注意：抠像是视频剪辑百分之百要掌握的技能，如果需要百分之百的抠像效果，拍摄就必须在绿幕环境或纯色环境中进行，才能最终实现百分之百的抠像效果。第三种：“遮罩”，“遮罩”包含5种具体的效果，但这五种效果都大同小异。它们依次是：绘制遮罩、渐变遮罩、图像遮罩、形状遮罩和晕影遮罩。其中“绘制遮罩”相对复杂，我们举例说明，默认的绘制方式是贝塞尔曲线。我们放一个片子在下方预览一下效果。可以看到被抹去的部分变成了透明，两个片段的画面融合在了一起。绘制完成以后还可以调整这个遮罩的其它属性参数，比如“羽化”和“散开”，使画面看起来更加柔和一些。19、音频的效果音频效果：为音频片段添加额外的音响效果。激活“效果浏览器窗口”，在窗口中可以看到，从上到下有8类音频效果，包括：EQ（均衡器）、回声、失真、电平、语音等一共一百余个具体的效果。使用方法跟视频的效果一样直接拖入到音频片段中即可。20、动画元素FinalCutPro可以是画面运动起来，但它本身无法制作动画元素。所以，我们要借助mac平台的另外一款应用-keynotekeynote是apple公司出品的幻灯片制作应用。我们可以通过它来快速的完成一些简单的动画。最后通过：文件-导出到-Quicktime-下一步-导出来到处一个1080p的动画视频，然后我们回到FinalCutPro中将刚才制作的动画视频导入进来。将它拖拽到时间线中，利用“抠像器”效果，将黑色背景抹去，这时候只留下动画元素了。21、音频增强音频增强主要包括了：响度、降噪、嗡嗡声消除音频增强的意义在于改善音质，尤其是录音和对话的音频中音频增强的作用是十分明显的22、色彩校正-颜色、饱和度、曝光我们看到的视频、mv等都是经过调色的，原始的素材在颜色、饱和度、曝光方面都有所不足。色彩校正是FinalCutPro重的术语，其实它也就是我们常说的“后期调色”。我们打开“校正增强选项”，选择“显示颜色版（command+6）”在“颜色”面板中，我们看到四个控制按钮，它们默认处于同一基准线。分别代表：全局、阴影、中间调、高光。对应的它们代表着画面中全局、暗部、中间部、亮部。把它们的控制按钮向上或向下拖移，拖移到特定的颜色区域时，画面中的颜色也就跟着有了变化。在“饱和度”面板中，我们看到有三哥控制按钮，它们默认处于同一基准线。分别代表：阴影、中间调、高光。对应的它们代表着画面中暗部、中间部、亮部。把它们的控制按钮向上或向下拖移，画面中的饱和度也就跟着有了变化。在“曝光”面板中，我们看到有三哥控制按钮，它们默认处于同一基准线。分别代表：阴影、中间调、高光。对应的它们代表着画面中暗部、中间部、亮部。把它们的控制按钮向上或向下拖移，画面中的曝光也就跟着有了变化。如果一个“色彩校正”不能满足需要，我们还可以点击添加“色彩校正”，多个“色彩校正”可以帮助我们实现更为复杂的调色效果。23、色彩校正--形状遮罩和颜色遮罩我们添加了“色彩校正”之后可以做进一步调整。每一个“色彩校正”我们都可以给它施加形状遮罩和颜色遮罩。遮罩：简单的理解就是将画面中的一部分留存，而将剩下的部分抹去。我们可以用鼠标调整遮罩，并发现遮罩内的“色彩校正”得到了保留，遮罩外的部分和原始的片段是一样的结论：给“色彩校正”施加“形状遮罩”，可以使遮罩中的调色得以保留，遮罩外的调色会被认定为失效。这就保证了我们可以给画面中特定的区域进行调色，而不影响其它区域的原始色彩。选择“颜色遮罩”-然后把鼠标点选到画布中，此时鼠标自动变为一个取色笔，我们可以在画面中点选颜色或框选颜色，取出一种颜色或临近的颜色范围，这个取出的颜色范围就成为我们的“颜色遮罩”的范围结论：施加了“颜色遮罩”的部位保留了“色彩校正”的调色，其他部位则恢复到了原素材的状态。“形状遮罩”和“颜色遮罩”是对“色彩校正”施加的两种不同的遮罩方式，它们控制着调色的范围，帮助我们实现对画面颜色的绝对掌握。即可以对画面中特定的部位进行调色，而其他部位则不受任何影响。其实“色彩校正”也是一种效果，在掌握了色彩校正的两种遮罩方法之后，回顾之前的内容，你会发现其实任何一种效果，都可以给他们施加形状遮罩和颜色遮罩，操作方法也都是一样的。这里重点来说是因为色彩校正最具代表性也是使用频率最高。24、FinalCutPro支持第三方的扩展包例如：iCastlePro©系列工具（2.0）等等，可以自己安装一些。只要是遵循apple公司的Motion规范即可。25、LUT：颜色查找吧LUT文件本质上是一种色彩数值的转换处理，可以将其理解为是一种预置的调色模板，通常由第三方厂商或专业人士设计制作。26、其他知识项FinalCutPro内置了许多种类的导出方式，建议在导出时直接选择“母版文件”即可。然后编码类型选择H.264（是目前比较主流的、在互联网传播中压缩质量最优的一种编码方式。）关于“资源库”需要提醒的是：资源库包含的是操作进程的数据，但并不包含素材文件本身，当我们将资源库搬迁到另一台电脑进行工作的时候，原来的素材文件也需要相应的路径搬迁，如果有移动工作的需求，可以在最早导入素材的时候，讲素材文件“拷贝到资源库”，这样就可以在搬迁的时候直接移动资源库文件就可以了，不用担心媒体文件的路径等等问题。但是它的问题在于可能会使你的资源库文件过大，难以控制以后的清理工作。关于缓存文件和数据清理，当所有的工作完成后，我们可以通过“文件”-删除生成的
